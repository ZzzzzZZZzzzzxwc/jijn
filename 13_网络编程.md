## 13.1 socket 基础

### 13.1.1 ★ TCP 客户端最小例子

```python
import socket

s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
s.connect(("example.com", 80))

# 发送 HTTP 请求
s.sendall(b"GET / HTTP/1.0\r\nHost: example.com\r\n\r\n")

# 接收响应
data = b""
while True:
    chunk = s.recv(4096)
    if not chunk:
        break
    data += chunk

s.close()
print(data[:200].decode())
```

**关键 API**：
- `socket()`：创建 socket
- `connect()`：连接服务器
- `sendall()`：保证全部发送（vs `send()` 可能只发部分）
- `recv(n)`：最多接收 n 字节（可能更少）
- `close()`：关闭

### 13.1.2 TCP 服务端最小例子

```python
import socket

server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
server.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)  # 允许端口复用
server.bind(("0.0.0.0", 8000))
server.listen(5)

print("listening on :8000")
while True:
    conn, addr = server.accept()
    print(f"connection from {addr}")
    
    data = conn.recv(1024)
    print(f"got: {data!r}")
    conn.sendall(b"HTTP/1.0 200 OK\r\n\r\nHello!")
    conn.close()
```

测试：浏览器访问 http://localhost:8000

### 13.1.3 UDP 示例

```python
# 服务端
import socket
s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
s.bind(("0.0.0.0", 9000))
data, addr = s.recvfrom(1024)
s.sendto(b"pong", addr)

# 客户端
s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
s.sendto(b"ping", ("server_ip", 9000))
data, _ = s.recvfrom(1024)
```

## 13.2 HTTP 客户端

### 13.2.1 标准库 http.client

```python
from http.client import HTTPSConnection

conn = HTTPSConnection("example.com")
conn.request("GET", "/")
resp = conn.getresponse()
print(resp.status, resp.reason)
print(resp.read()[:200])
conn.close()
```

### 13.2.2 ★ requests（最推荐）

```python
import requests

r = requests.get("https://api.github.com/users/python", timeout=10)
print(r.status_code)
print(r.json())            # 自动解析 JSON

# POST
r = requests.post("https://api.example.com/users",
                  json={"name": "Alice"},
                  headers={"Authorization": "Bearer xxx"},
                  timeout=10)

# Session（连接池 + 共享 cookies）
with requests.Session() as s:
    s.headers.update({"User-Agent": "myapp/1.0"})
    r = s.get(url1)
    r = s.get(url2)        # 复用连接
```

### 13.2.3 ⚠️ 必须设 timeout

```python
# ❌ 没 timeout，可能永远卡住
r = requests.get(url)

# ✅
r = requests.get(url, timeout=10)         # 整体 10 秒
r = requests.get(url, timeout=(3, 30))    # connect 3s, read 30s
```

### 13.2.4 重试

```python
from requests.adapters import HTTPAdapter
from urllib3.util.retry import Retry

retry = Retry(total=3, backoff_factor=1,
              status_forcelist=[500, 502, 503, 504])
adapter = HTTPAdapter(max_retries=retry)

s = requests.Session()
s.mount("https://", adapter)
s.mount("http://", adapter)

r = s.get(url, timeout=10)
```

## 13.3 异步 HTTP：aiohttp

```python
import asyncio
import aiohttp

async def fetch(session, url):
    async with session.get(url, timeout=aiohttp.ClientTimeout(total=10)) as r:
        return await r.text()

async def main():
    async with aiohttp.ClientSession() as session:
        results = await asyncio.gather(*[
            fetch(session, f"https://example.com/{i}")
            for i in range(100)
        ])
    print(f"got {len(results)} responses")

asyncio.run(main())
```

更现代的选择：**httpx**（同时支持同步和异步）：

```python
import httpx

# 同步
r = httpx.get("https://...")

# 异步
async with httpx.AsyncClient() as client:
    r = await client.get("https://...")
```

## 13.4 简易 HTTP 服务器

### 13.4.1 标准库 http.server

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
import json

class Handler(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path == "/":
            self.send_response(200)
            self.send_header("Content-Type", "application/json")
            self.end_headers()
            self.wfile.write(json.dumps({"hello": "world"}).encode())
        else:
            self.send_error(404)
    
    def do_POST(self):
        length = int(self.headers["Content-Length"])
        body = self.rfile.read(length)
        data = json.loads(body)
        # 处理 data ...
        self.send_response(201)
        self.end_headers()

if __name__ == "__main__":
    HTTPServer(("0.0.0.0", 8080), Handler).serve_forever()
```

适合内部工具、demo。生产用 Flask/FastAPI。

### 13.4.2 FastAPI（现代异步框架）

## 13.5 WebSocket

### 13.5.1 客户端（websockets 库）

```python
import asyncio
import websockets

async def chat():
    async with websockets.connect("ws://localhost:8765") as ws:
        await ws.send("hello")
        async for message in ws:
            print(f"received: {message}")
            if message == "bye":
                break

asyncio.run(chat())
```

### 13.5.2 服务端

```python
import asyncio
import websockets

async def handler(ws):
    async for message in ws:
        await ws.send(f"echo: {message}")

async def main():
    async with websockets.serve(handler, "0.0.0.0", 8765):
        await asyncio.Future()    # 永远运行

asyncio.run(main())
```

## 13.7 常见误区

### 误区 1：忘记关 socket

```python
# ❌ 用完不关，泄漏 fd
s = socket.socket(...)
s.connect(...)
# ... 用完没 close

# ✅ 用 with（Python 3.4+）
with socket.create_connection(("example.com", 80)) as s:
    s.sendall(b"...")
```

### 误区 2：以为 recv(n) 一定收 n 字节

```python
# ❌
data = s.recv(1024)    # 可能只收到 100 字节

# ✅ 循环直到收齐
data = b""
while len(data) < expected_size:
    chunk = s.recv(min(4096, expected_size - len(data)))
    if not chunk:
        raise ConnectionError("closed")
    data += chunk
```

### 误区 3：HTTPS 证书验证

```python
# ❌ 关闭验证（除调试外都是漏洞）
requests.get(url, verify=False)

# ✅ 自定义 CA
requests.get(url, verify="/path/to/ca.pem")
```

### 误区 4：把 requests 用在 async 函数

```python
async def bad():
    requests.get(url)    # 阻塞整个事件循环！

async def good():
    async with httpx.AsyncClient() as c:
        await c.get(url)
```

## 13.8 面试常问

### Q1：TCP 和 UDP 的区别？什么场景用 UDP？

**答**：TCP 面向连接、有序、可靠、有流控；UDP 无连接、不保证顺序、不重传。
- TCP：HTTP、SSH、FTP（绝大多数）
- UDP：DNS、视频流、游戏（实时性优先）

### Q2：requests 和 aiohttp 的区别？

**答**：requests 是同步 HTTP 库，aiohttp 是异步。需要并发上百请求时用 aiohttp。**注意：不要在 async 函数里用 requests**。

### Q3：HTTP 长连接是什么？

**答**：HTTP/1.1 默认 Keep-Alive，TCP 连接复用多个 HTTP 请求，避免握手开销。`requests.Session` 自动用连接池。

### Q4：什么是 SO_REUSEADDR？

**答**：允许重启服务器时立刻复用刚被关闭的端口（避免 TIME_WAIT 状态导致 bind 失败）。

### 练习 13.2（中级）

写一个 API 健康检查器：并发探测多个 URL，返回每个的状态码和响应时间，超时 5 秒。

<details>
<summary>答案</summary>

```python
import asyncio, time
import httpx

async def check(client, url):
    start = time.time()
    try:
        r = await client.get(url, timeout=5)
        return {"url": url, "status": r.status_code, "elapsed": time.time() - start}
    except Exception as e:
        return {"url": url, "error": str(e), "elapsed": time.time() - start}

async def health_check(urls):
    async with httpx.AsyncClient() as client:
        return await asyncio.gather(*[check(client, u) for u in urls])

# 用法
results = asyncio.run(health_check(["https://example.com", "https://api.github.com"]))
```
</details>
