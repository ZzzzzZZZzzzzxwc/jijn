## 知识体系总图

```
Python 基础
├── 数据类型
│   ├── 数值: int, float, bool, complex
│   ├── 序列: str, list, tuple
│   ├── 映射: dict
│   └── 集合: set, frozenset
├── 控制流
│   ├── 条件: if / elif / else / match-case
│   ├── 循环: for / while / break / continue
│   └── 异常: try / except / finally / raise
├── 函数
│   ├── 定义与参数 (*args, **kwargs)
│   ├── 作用域 (LEGB)
│   ├── 闭包与装饰器
│   ├── 生成器 (yield)
│   └── lambda 与高阶函数
├── 面向对象
│   ├── 类与实例
│   ├── 继承与多态
│   ├── 魔术方法
│   ├── property 与描述符
│   └── 抽象类与 dataclass
├── 模块与包
│   ├── import 机制
│   ├── 标准库
│   └── 虚拟环境与 pip
└── 高级特性
    ├── 迭代器协议
    ├── 上下文管理器 (with)
    ├── 正则表达式 (re)
    ├── 并发 (threading / multiprocessing / asyncio)
    └── 类型提示 (typing)
```

## 怎么用这个仓库

1. **手机党**：直接在 GitHub 上点开 `.md` 文件看，自动渲染
2. **电脑党**：克隆到本地，用 VS Code / Typora 打开
3. **碎片时间**：打开速查手册，随机翻到一个章节复习
4. **遇到问题**：先查分类笔记对应章节，找代码示例

## 学习建议

- ✅ 每天至少看 15 分钟
- ✅ 看完一个知识点，立刻打开 Python 敲一遍
- ✅ 遇到不懂的，先敲代码看结果，再理解原理
- ✅ 第一册至少过 3 遍再进入第二册
- ❌ 不要只看不敲
- ❌ 不要贪多，一天一个小节就够了
- ❌ 不要跳过基础直接学框架
